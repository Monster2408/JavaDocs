すべてのクラス

  • すべてのクラス インタフェースの概要 クラスの概要 列挙型の概要 
    クラス 説明
    Advancement
    A configurable advancement which later may be saved to a file, loaded into the server.
    Advancement.Frame
    Frame styles, displayed on the advancement screen around advancements.
    AdvancementFactory
    A class containing methods which should drastically speed up the creation of the common advancements.
    BlockObject
    Specifies information about a block.
    BredAnimalsTrigger
    Fires whenever two animals are bred and a child is born.
    BrewedPotionTrigger
    Fires whenever any (!)
    ChangedDimensionTrigger
    Fires whenever the player switches to another dimension.
    ConstructBeaconTrigger
    Fires whenever the player is close enough to a beacon and the beacon ticks, possibly giving the player status effects (depending on whether any effect is selected in the beacon).
    ConsumeItemTrigger
    Fires whenever the player eats food, drinks milk or a potion/water bottle.
    CuredZombieVillagerTrigger
    Fires whenever a zombie villager turns into a villager.
    DamageFlagsObject
    Specifies information about a damage event, similarly to DamageObject.
    DamageObject
    Specifies information about a damage event.
    DeathObject
    Specifies information about a death event.
    Dimension
    Specifies a world type.
    DistanceObject
    Specifies information about a distance between two points.
    Effect
    Specifies a status effect.
    EffectObject
    Specifies information about a single applied status effect.
    EffectsChangedTrigger
    Fires whenever the player receives or loses a status effect.
    Enchant
    Specifies an item enchantment.
    EnchantedItemTrigger
    Fires whenever the player enchants an item.
    EnchantObject
    Specifies information about an enchantment applied to an item.
    EnterBlockTrigger
    Fires whenever a hitbox intersects with a block, including air, as well as the block the an ender pearl had landed at.
    EntityHurtPlayerTrigger
    Fires whenever whenever the player takes damage, even if that damage is 0 or blocked.
    EntityKilledPlayerTrigger
    Fires whenever any mob, excluding players kills the player.
    EntityObject
    Specifies information about an entity.
    Feature
    Specifies a structure in a world.
    ImpossibleTrigger
    Never fires, therefore this trigger cannot be fulfilled, except if it's explicitly granted using commands and such.
    InventoryChangedTrigger
    Fires whenever items are added to or removed from the player's inventory.
    ItemDurabilityChangedTrigger
    Fires whenever any item's durability in the player's inventory changes.
    ItemObject
    Specifies information about an item.
    JsonBuilder
    JsonObject creation utility, used internally.
    LevitationTrigger
    Fires whenever the player is under the levitation status effect.
    LocationObject
    Specifies information about a location in any world.
    LocationTrigger
    Fires every 20 ticks (every second).
    MaterialId
    Specifies an item/block type.
    MobType
    Specifies an entity type.
    NetherTravelTrigger
    Fires whenever the player enters the nether and then returns to the overworld.
    PlacedBlockTrigger
    Fires whenever the player places a block.
    PlayerHurtEntityTrigger
    Fires whenever a player deals any amount of damage (including 0) to an entity (excluding players).
    PlayerKilledEntityTrigger
    Fires whenever a player kills a living entity, except players.
    Potion
    Specifies a potion's effect.
    Potion.Type
    The type of the potion item.
    RangeObject
    Specifies a range of values.
    RecipeUnlockedTrigger
    Fires whenever the player unlocks a recipe.
    Rewards
    Specifies rewards which are given when the advancement is completed.
    SharedEnum
    Enums implementing this interface are used to define data usually in SharedObjects.
    SharedObject
    Objects extending this class are used to define data, eg. icons, criteria of triggers and can be converted to JSON elements.
    SleptInBedTrigger
    Fires when the player successfully enters a bed (the player does not have to pass the night).
    StatusEffectsObject
    Specifies multiple applied status effects.
    SummonedEntityTrigger
    Fires whenever the player creates an entity.
    TameAnimalTrigger
    Fires whenever the player tames an animal.
    TickTrigger
    Fires every tick.
    Trigger
    A trigger of an advancement.
    Trigger.Type
    The type of trigger, this determines which event the trigger listens to.
    UsedEnderEyeTrigger
    Fires whenever the player uses an Eye of Ender.
    UsedTotemTrigger
    Fires whenever the player uses a Totem of Undying.
    Validator
    Validation utility, used internally.
    VillagerTradeTrigger
    Fires whenever the player completes a trade with a villager.
