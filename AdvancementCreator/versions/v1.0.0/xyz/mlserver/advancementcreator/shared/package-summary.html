パッケージ xyz.mlserver.advancementcreator.shared

AdvancementCreator's shared package
  • インタフェースの概要 
    インタフェース 説明
    SharedEnum
    Enums implementing this interface are used to define data usually in SharedObjects.
  • クラスの概要 
    クラス 説明
    BlockObject
    Specifies information about a block.
    DamageFlagsObject
    Specifies information about a damage event, similarly to DamageObject.
    DamageObject
    Specifies information about a damage event.
    DeathObject
    Specifies information about a death event.
    DistanceObject
    Specifies information about a distance between two points.
    EffectObject
    Specifies information about a single applied status effect.
    EnchantObject
    Specifies information about an enchantment applied to an item.
    EntityObject
    Specifies information about an entity.
    ItemObject
    Specifies information about an item.
    LocationObject
    Specifies information about a location in any world.
    RangeObject
    Specifies a range of values.
    SharedObject
    Objects extending this class are used to define data, eg. icons, criteria of triggers and can be converted to JSON elements.
    StatusEffectsObject
    Specifies multiple applied status effects.
  • 列挙型の概要 
    列挙 説明
    Dimension
    Specifies a world type.
    Effect
    Specifies a status effect.
    Enchant
    Specifies an item enchantment.
    Feature
    Specifies a structure in a world.
    MaterialId
    Specifies an item/block type.
    MobType
    Specifies an entity type.
    Potion
    Specifies a potion's effect.
    Potion.Type
    The type of the potion item.
