      xyz.mlserver.advancementcreator.triggerでのTriggerのサブクラス 
      修飾子とタイプ クラス 説明
      class  BredAnimalsTrigger
      Fires whenever two animals are bred and a child is born.
      class  BrewedPotionTrigger
      Fires whenever any (!)
      class  ChangedDimensionTrigger
      Fires whenever the player switches to another dimension.
      class  ConstructBeaconTrigger
      Fires whenever the player is close enough to a beacon and the beacon ticks, possibly giving the player status effects (depending on whether any effect is selected in the beacon).
      class  ConsumeItemTrigger
      Fires whenever the player eats food, drinks milk or a potion/water bottle.
      class  CuredZombieVillagerTrigger
      Fires whenever a zombie villager turns into a villager.
      class  EffectsChangedTrigger
      Fires whenever the player receives or loses a status effect.
      class  EnchantedItemTrigger
      Fires whenever the player enchants an item.
      class  EnterBlockTrigger
      Fires whenever a hitbox intersects with a block, including air, as well as the block the an ender pearl had landed at.
      class  EntityHurtPlayerTrigger
      Fires whenever whenever the player takes damage, even if that damage is 0 or blocked.
      class  EntityKilledPlayerTrigger
      Fires whenever any mob, excluding players kills the player.
      class  ImpossibleTrigger
      Never fires, therefore this trigger cannot be fulfilled, except if it's explicitly granted using commands and such.
      class  InventoryChangedTrigger
      Fires whenever items are added to or removed from the player's inventory.
      class  ItemDurabilityChangedTrigger
      Fires whenever any item's durability in the player's inventory changes.
      class  LevitationTrigger
      Fires whenever the player is under the levitation status effect.
      class  LocationTrigger
      Fires every 20 ticks (every second).
      class  NetherTravelTrigger
      Fires whenever the player enters the nether and then returns to the overworld.
      class  PlacedBlockTrigger
      Fires whenever the player places a block.
      class  PlayerHurtEntityTrigger
      Fires whenever a player deals any amount of damage (including 0) to an entity (excluding players).
      class  PlayerKilledEntityTrigger
      Fires whenever a player kills a living entity, except players.
      class  RecipeUnlockedTrigger
      Fires whenever the player unlocks a recipe.
      class  SleptInBedTrigger
      Fires when the player successfully enters a bed (the player does not have to pass the night).
      class  SummonedEntityTrigger
      Fires whenever the player creates an entity.
      class  TameAnimalTrigger
      Fires whenever the player tames an animal.
      class  TickTrigger
      Fires every tick.
      class  UsedEnderEyeTrigger
      Fires whenever the player uses an Eye of Ender.
      class  UsedTotemTrigger
      Fires whenever the player uses a Totem of Undying.
      class  VillagerTradeTrigger
      Fires whenever the player completes a trade with a villager.
